OPERATEURS:

a) Operateurs Arithmetique

    +, -, *, /, %, //, **, 

    ** : midika hoe exposant 
    // : midika hoe division entière
    % : modulo ampiasaina aminy mitady reste de division.

b) Operateurs de comparaison :

    >, >=, <, <=, ==, !=, <>
     Fanazavana kely :
                    <> : midika hoe different de ...

c) Operateurs logiques

    ET, OU, NON,(pseudo code); and, or, not(code Python)

d) Operateurs d'affectation : 

    Exemple directe : 
                    += : midka hoe miantomboka avy ambany miakatra (ex: 1,2,3,4,5,6,7)(n ← n+1 : pseudo code)
                    -= : midika hoe miantomboka avy ambony midina(ex: 7,6,5,4,3,2,1)(n ← n-1 : pseudo code)
                     = : affectation mitovy aminy "←"(pseudo code)

e) Operateurs d'incrementation : 

    ++ et -- aminy langage C, C++ mitovy aminy += ; -=
